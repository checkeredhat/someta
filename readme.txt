So Meta is the easiest way to add meta tags to your Wordpress site. This plugin adds Open Graph Meta Tags for Facebook and Google. It also adds meta information for Twitter Cards.
This plugin is absolutely, 100% free and open-source (GPLv2).

**How to Use**
Install and activate the plugin. Navigate to the So Meta admin menu. Add your default description, keywords, and thumbnail image url.

**Need help?**
You can get help or leave feature requests by sending me an email:

**Features**
- Adds Open Graph Meta Tags for Facebook and Google.
- Adds meta information for Twitter Cards.
- Adds default meta description and keywords.
- Easy for developers to modify.
